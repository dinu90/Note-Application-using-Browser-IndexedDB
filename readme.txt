1. Adding Note:
	Yellow tab is used to add a Note in the indexed DB.
2. List View:
	All the notes from the Indexed DB will be displayed in a table with a delete option in the blue colored note at the bottom of the screen. On selecting the subject, the detailed view of the note will be displayed in the purple tab.

3. Detailed View:
	The selected note details like who has created, when the note was created (i.e, date with time), subject and Message will be displayed in the purple tab at the bottom of the screen. Delete button is used to delete the note from the DB.
